# Extensão Marketing

Esta extensão adiciona capacidades de marketing ao Sleep Agent, incluindo copywriting, estratégia e tráfego pago.

## Agentes Disponíveis

### Copywriter (`/copy`)
Especialista em textos persuasivos. Use para:
- Headlines e títulos
- Textos de venda
- Emails marketing
- Bullet points
- Scripts de vídeo

### Estrategista (`/estrategia`)
Especialista em estratégia de marketing. Use para:
- Definir posicionamento
- Criar ofertas
- Analisar público-alvo
- Definir proposta de valor
- Precificação

### Tráfego (`/ads`)
Especialista em tráfego pago. Use para:
- Criar anúncios
- Definir segmentação
- Estruturar campanhas
- Otimizar resultados

## Tarefas Prontas

| Tarefa | Comando | O que faz |
|--------|---------|-----------|
| Criar Oferta | `/criar-oferta` | Guia passo a passo para construir oferta irresistível |
| Criar Headline | `/criar-headline` | Gera headlines usando fórmulas comprovadas |
| Criar Email | `/criar-email` | Escreve emails de venda estruturados |

## Workflows

| Workflow | Comando | O que faz |
|----------|---------|-----------|
| Lançamento | `/lancamento` | Prepara todos os elementos de um lançamento |
| Campanha | `/campanha` | Cria campanha de marketing completa |

## Base de Conhecimento

A extensão inclui conhecimento sobre:
- **Frameworks de copywriting**: AIDA, PAS, BAB, 4Us
- **Gatilhos mentais**: Escassez, autoridade, prova social, etc.
- **Fórmulas de headlines**: 50+ fórmulas testadas

## Exemplos

Veja exemplos reais em `exemplos/`:
- `headline-exemplo.md` - Headlines criadas para diferentes nichos
- `email-exemplo.md` - Emails de venda completos

## Personalizando

Você pode adicionar:
- Novos exemplos na pasta `exemplos/`
- Conhecimento específico do seu nicho em `conhecimento/`
- Novas tarefas copiando templates existentes em `tarefas/`
