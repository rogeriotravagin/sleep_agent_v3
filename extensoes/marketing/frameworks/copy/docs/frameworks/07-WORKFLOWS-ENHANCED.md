## MASTER DECISION MATRIX

Before starting any workflow, answer these 5 questions:

```
┌─────────────────────────────────────────────────────────────────────────┐
│ Q1: AWARENESS LEVEL - Where is your audience?                           │
│     □ Unaware      □ Problem Aware    □ Solution Aware                  │
│     □ Product Aware □ Most Aware                                        │
├─────────────────────────────────────────────────────────────────────────┤
│ Q2: MARKET SOPHISTICATION - How crowded is the market?                  │
│     □ Stage 1 (Be first)    □ Stage 2 (Be bigger)                       │
│     □ Stage 3 (Mechanism)   □ Stage 4 (New mechanism)                   │
│     □ Stage 5 (Prospect identification)                                 │
├─────────────────────────────────────────────────────────────────────────┤
│ Q3: PRICE POINT - What's the investment level?                          │
│     □ Low (<$100)   □ Mid ($100-$1000)   □ High ($1000-$5000)           │
│     □ Premium ($5000+)                                                  │
├─────────────────────────────────────────────────────────────────────────┤
│ Q4: TRAFFIC TEMPERATURE - How do they arrive?                           │
│     □ Cold (never heard of you)    □ Warm (some familiarity)            │
│     □ Hot (ready to buy)                                                │
├─────────────────────────────────────────────────────────────────────────┤
│ Q5: PRIMARY OUTPUT - What are you creating?                             │
│     □ VSL    □ Sales Letter    □ Email Sequence    □ Ad Creative        │
│     □ Webinar    □ Direct Mail    □ Landing Page    □ Content           │
└─────────────────────────────────────────────────────────────────────────┘
```

## WORKFLOW 1: FULL PRODUCT LAUNCH (5-7 Days)

**Best for:** Course launches, membership openings, new product releases
**Expected output:** Complete funnel with VSL/Sales Letter + Email sequence

```
═══════════════════════════════════════════════════════════════════════════
DAY 1: FOUNDATION (TIER 0) - Never Skip This
═══════════════════════════════════════════════════════════════════════════

STEP 1: Scientific Framework Setup (Claude Hopkins)
┌─────────────────────────────────────────────────────────────────────────┐
│ Hopkins invented split testing, coupon tracking, and test marketing.    │
│ His Scientific Advertising is "the foundation of direct marketing."     │
│                                                                         │
│ □ Define what will be measured (primary KPI)                            │
│ □ Set up tracking mechanisms before writing                             │
│ □ Establish success criteria (what number = winner?)                    │
│ □ Plan A/B test variations from the start                               │
│ □ Remember: "The only purpose of advertising is to make sales"          │
└─────────────────────────────────────────────────────────────────────────┘

STEP 2: Awareness Diagnosis (Eugene Schwartz)
┌─────────────────────────────────────────────────────────────────────────┐
│ The 5 Levels of Awareness determine EVERYTHING else:                    │
│                                                                         │
│ LEVEL           │ THEY KNOW...           │ YOUR JOB IS TO...            │
│ ────────────────┼────────────────────────┼─────────────────────────────│
│ Unaware         │ Nothing                │ Connect to existing desire   │
│ Problem Aware   │ They have a problem    │ Show your solution exists    │
│ Solution Aware  │ Solutions exist        │ Show yours is different      │
│ Product Aware   │ Your product exists    │ Prove it's best for them     │
│ Most Aware      │ Everything             │ Just make the offer          │
│                                                                         │
│ □ Survey/interview 5-10 prospects                                       │
│ □ Map where MAJORITY of your audience sits                              │
│ □ This determines copy LENGTH and APPROACH                              │
│                                                                         │
│ AWARENESS → COPY LENGTH:                                                │
│ Unaware = 5000+ words │ Most Aware = 500 words                          │
└─────────────────────────────────────────────────────────────────────────┘

STEP 3: Mental Conversation Entry (Robert Collier)
┌─────────────────────────────────────────────────────────────────────────┐
│ "Always enter the conversation already taking place in the              │
│  customer's mind." - Collier (cited by Halbert, Kennedy, Carlton)       │
│                                                                         │
│ The 6 Prime Motives (identify which are active):                        │
│ □ Love          □ Gain/Greed      □ Duty                                │
│ □ Pride         □ Self-indulgence □ Self-preservation                   │
│                                                                         │
│ Questions to answer:                                                    │
│ □ What conversation is ALREADY in their head right now?                 │
│ □ What recent event/trend can you hook into?                            │
│ □ What's their dominant emotion about this topic?                       │
│ □ How do you "enter" without jarring them?                              │

│                                                                         │

│    □ Discovery moment                                                   │

│ □ Paint vivid picture of life AFTER they buy                            │
│ □ Use present tense ("You wake up... You feel... You see...")           │
│ □ Include specific sensory details                                      │
│ □ Address their deepest desire                                          │
│                                                                         │
│ PAIN AMPLIFICATION (ethical):                                           │
│ □ Show cost of inaction (not just benefit of action)                    │
│ □ "Every day you wait, [consequence continues/worsens]"                 │
│ □ Make status quo uncomfortable                                         │
│ □ Create urgency through implication, not manipulation                  │

```
