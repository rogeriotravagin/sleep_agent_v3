## SECTION 1: ONE IDEA PER SENTENCE

### Sentence Length Check

- [ ] Nenhuma sentença tem mais de 20 palavras
- [ ] Média de palavras por sentença é ≤15
- [ ] Sentenças curtas (≤10 palavras) são maioria
- [ ] Frases longas foram divididas estrategicamente

### Sentence Structure Check

- [ ] Cada sentença expressa apenas UMA ideia
- [ ] Nenhuma sentença tem múltiplos "e" conectando ideias completas
- [ ] Nenhuma sentença tem "mas" no meio conectando ideias opostas
- [ ] Nenhuma sentença requer releitura para entender

### Fragmentation Check

- [ ] Fragmentos usados estrategicamente para ênfase
- [ ] Fragmentos soam naturais, não forçados
- [ ] Fragmentos seguem ideias completas (para expansão)

## SECTION 2: CONVERSATIONAL TONE

### Natural Language Check

- [ ] Usa contrações naturalmente (você, não, é, está)
- [ ] Evita linguagem formal/acadêmica
- [ ] Soa como uma pessoa falando, não um documento
- [ ] Usa gírias apropriadas ao público (quando relevante)

### Direct Address Check

- [ ] Fala diretamente com "você" (não "os clientes" ou "pessoas")
- [ ] Usa perguntas diretas ao leitor
- [ ] Estabelece conexão pessoal
- [ ] Evita terceira pessoa quando possível

### Read-Aloud Test

- [ ] Lido em voz alta sem tropeços
- [ ] Não fica sem ar em nenhuma sentença
- [ ] Sons natural, não robótico
- [ ] Flui como conversa real

### Bar Test

- [ ] Você diria isso para um amigo num bar?
- [ ] Não soa pretensioso ou corporativo?
- [ ] Parece autêntico, não fabricado?
- [ ] Você não teria vergonha de falar assim?

## SECTION 3: JARGON ELIMINATION

### Buzzword Check

- [ ] Nenhum uso de "utilizar" (usar "usar")
- [ ] Nenhum uso de "implementar" (usar "fazer", "aplicar")
- [ ] Nenhum uso de "alavancar" (usar "usar")
- [ ] Nenhum uso de "otimizar" (usar "melhorar")
- [ ] Nenhum uso de "sinergia" (usar "trabalho em equipe")
- [ ] Nenhum uso de "paradigma" (usar "forma de pensar")

### Corporate Speak Check

- [ ] Sem "solução" quando significa "produto"
- [ ] Sem "plataforma" quando significa "ferramenta"
- [ ] Sem "escalável" quando significa "que cresce"
- [ ] Sem "disruptivo" quando significa "diferente"
- [ ] Sem "holístico" quando significa "completo"
- [ ] Sem "stakeholders" quando significa "pessoas envolvidas"

### Unnecessary Complexity Check

- [ ] Sem frases em latim ou estrangeiras desnecessárias
- [ ] Sem siglas não explicadas
- [ ] Sem termos técnicos quando alternativas simples existem
- [ ] Sem palavras de mais de 3 sílabas quando uma menor funciona

### Simplicity Test

- [ ] Um estudante do 8º ano entenderia?
- [ ] Seu avô entenderia?
- [ ] Um não-especialista entenderia?
- [ ] Alguém apressado entenderia?

## SECTION 4: HOOK & HEADLINE

### Headline Power Check

- [ ] Para o leitor IMEDIATAMENTE?
- [ ] Promessa específica e desejável?
- [ ] Cria curiosidade irresistível?
- [ ] Tem elemento "one-legged golfer"?
- [ ] Funcionaria numa timeline barulhenta?

### Hook Mining Check

- [ ] O "one-legged golfer" foi identificado?
- [ ] O hook é inesperado ou contra-intuitivo?
- [ ] O hook é visual/sensorial?
- [ ] O hook é específico (não genérico)?
- [ ] O hook gera "espera, o quê?"?

### Specificity Check

- [ ] Números específicos presentes?
- [ ] Timeframes específicos?
- [ ] Resultados específicos prometidos?
- [ ] Detalhes que tornam real?

## SECTION 5: LIZARD BRAIN APPEAL

### Primal Trigger Check

- [ ] Pelo menos um gatilho primal presente:
  - [ ] Sobrevivência/Medo
  - [ ] Ganância/Desejo de ganho
  - [ ] Luxúria/Atração
  - [ ] Vaidade/Status
  - [ ] Preguiça/Facilidade
  - [ ] Curiosidade

### Emotional Sequence Check

- [ ] Emoção aparece ANTES de lógica?
- [ ] Leitor SENTE algo nas primeiras 3 frases?
- [ ] A emoção é intensificada (agitação)?
- [ ] A solução proporciona alívio emocional?

### Visceral Language Check

- [ ] Linguagem sensorial presente (ver, ouvir, sentir)?
- [ ] Descrições vívidas e visuais?
- [ ] "Future pacing" com detalhes sensoriais?
- [ ] Emoções nomeadas explicitamente?

### Fear/Desire Balance

- [ ] Medo de perda presente?
- [ ] Desejo de ganho presente?
- [ ] Balanço apropriado entre os dois?
- [ ] Urgência emocional criada?

## SECTION 6: STORY & PROOF

### Real Story Check

- [ ] Pelo menos uma história real (não ficção) presente?
- [ ] A história tem:
  - [ ] Pessoa específica (nome ou descriptor)
  - [ ] Situação específica (antes)
  - [ ] Transformação específica (depois)
  - [ ] Detalhes que tornam real

### Testimonial Quality Check

- [ ] Testemunhos têm resultados específicos?
- [ ] Testemunhos incluem ceticismo inicial?
- [ ] Testemunhos têm detalhes verificáveis?
- [ ] Testemunhos usam linguagem real (não polida demais)?

### Proof Stack Check

- [ ] Claims têm prova adjacente?
- [ ] Números específicos citados?
- [ ] Fontes mencionadas quando relevante?
- [ ] Credibilidade estabelecida?

## SECTION 7: CTA & CLOSE

### Clarity Check

- [ ] O leitor sabe EXATAMENTE o que fazer?
- [ ] O próximo passo é óbvio?
- [ ] Não há ambiguidade na ação?
- [ ] Instruções são simples e diretas?

### Urgency Check

- [ ] Razão para agir AGORA presente?
- [ ] Urgência é justificada (não fake)?
- [ ] Deadline ou escassez específica?
- [ ] Consequência de não agir clara?

### Risk Reversal Check

- [ ] Garantia presente?
- [ ] Garantia remove risco do comprador?
- [ ] Garantia é clara e sem "letras miúdas"?
- [ ] Garantia é memorável (tem nome)?

### CTA Visibility Check

- [ ] CTA impossível de perder?
- [ ] CTA repetido se copy for longa?
- [ ] CTA usa linguagem de ação (verbo)?
- [ ] CTA tem micro-copy de suporte?

## SECTION 8: FINAL VALIDATION

### The Carlton Questions

- [ ] "Um estudante do 8º ano entenderia isso?"
- [ ] "Eu diria isso para um amigo num bar?"
- [ ] "Li em voz alta sem tropeçar?"
- [ ] "Cada palavra ganha seu lugar?"
- [ ] "Encontrei o one-legged golfer?"
- [ ] "O leitor SENTE algo antes de PENSAR?"
- [ ] "O próximo passo é óbvio?"

### Red Flags Check

- [ ] SEM sentenças com mais de 25 palavras?
- [ ] SEM jargão corporativo?
- [ ] SEM headline sem hook?
- [ ] SEM abertura sem emoção?
- [ ] SEM CTA vago ou ausente?
- [ ] SEM claims sem prova?

### Excellence Indicators

- [ ] Flui como conversa natural?
- [ ] Cada sentença puxa para a próxima?
- [ ] Hook é inesperado e memorável?
- [ ] Emoção é palpável?
- [ ] Leitura em voz alta é prazerosa?

## QUICK REFERENCE

### Carlton's Golden Rules

1. **One idea per sentence** - Se tem mais de uma, divida
2. **Write like you talk** - Se não diria assim, não escreva
3. **Find the one-legged golfer** - O hook está escondido
4. **Hit lizard brain first** - Emoção antes de lógica
5. **Every word earns its place** - Corte o resto
6. **Read it out loud** - Se tropeçar, reescreva
7. **Keep it simple** - 8º ano deve entender

### Instant Fixes

| Problema | Fix Imediato |
|----------|--------------|
| Sentença longa | Divida no "e" ou "mas" |
| Jargão | Substitua por palavra simples |
| Sem hook | Procure o detalhe inesperado |
| Sem emoção | Adicione medo ou desejo |
| CTA fraco | "Clique agora" + urgência |
| Sem prova | Adicione número ou história |

### Warning Signs (Stop & Fix)

- ❌ Qualquer sentença > 25 palavras
- ❌ Qualquer jargão corporativo
- ❌ Headline que não para
- ❌ Abertura sem emoção
- ❌ CTA que pode ser ignorado
- ❌ Claim sem prova

### Excellence Markers

- ✅ Lê como conversa natural
- ✅ Cada sentença puxa para a próxima
- ✅ Hook memorável e inesperado
- ✅ Emoção palpável do início ao fim
- ✅ Ação impossível de ignorar
