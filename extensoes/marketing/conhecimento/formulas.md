## Como Usar Este Documento

1. **Escolha 3-5 fórmulas** que se encaixam no contexto
2. **Adapte** substituindo os elementos pelos seus
3. **Crie variações** de cada uma
4. **Teste** para ver qual performa melhor

**Dica:** Combine elementos de diferentes fórmulas para criar headlines únicas.
