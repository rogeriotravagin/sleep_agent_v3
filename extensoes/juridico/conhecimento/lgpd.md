# Base de Conhecimento: LGPD

## Lei Geral de Protecao de Dados (Lei 13.709/2018)

### Conceitos Fundamentais

| Termo | Definicao |
|-------|-----------|
| **Dado Pessoal** | Informacao relacionada a pessoa natural identificada ou identificavel |
| **Dado Sensivel** | Origem racial, conviccao religiosa, opiniao politica, saude, vida sexual, dado genetico ou biometrico |
| **Titular** | Pessoa natural a quem os dados se referem |
| **Controlador** | Quem decide sobre o tratamento dos dados |
| **Operador** | Quem realiza o tratamento em nome do controlador |
| **Encarregado (DPO)** | Responsavel por comunicacao com titulares e ANPD |
| **Tratamento** | Qualquer operacao com dados pessoais |
| **ANPD** | Autoridade Nacional de Protecao de Dados |

---

## Bases Legais para Tratamento (Art. 7)

### 1. Consentimento
- Livre, informado e inequivoco
- Para finalidade especifica
- Pode ser revogado a qualquer momento
- Cuidado: e a base mais fragil

### 2. Cumprimento de Obrigacao Legal
- Quando a lei exige o tratamento
- Ex: Guarda de documentos fiscais, eSocial

### 3. Execucao de Politicas Publicas
- Apenas para administracao publica

### 4. Estudos por Orgao de Pesquisa
- Preferencialmente com anonimizacao

### 5. Execucao de Contrato
- Necessario para cumprir contrato com o titular
- Ex: Dados para entrega de produto comprado

### 6. Exercicio Regular de Direitos
- Em processo judicial, administrativo ou arbitral

### 7. Protecao da Vida
- Emergencias de saude

### 8. Tutela da Saude
- Apenas por profissionais de saude

### 9. Interesse Legitimo
- Do controlador ou terceiro
- Nao pode prevalecer sobre direitos do titular
- Requer documentacao (LIA - Legitimate Interest Assessment)

### 10. Protecao do Credito
- Analise de credito conforme legislacao

---

## Direitos do Titular (Art. 18)

1. **Confirmacao** - Saber se existe tratamento
2. **Acesso** - Obter copia dos dados
3. **Correcao** - Corrigir dados incorretos
4. **Anonimizacao/Bloqueio/Eliminacao** - De dados desnecessarios
5. **Portabilidade** - Transferir para outro fornecedor
6. **Eliminacao** - Dos dados tratados com consentimento
7. **Informacao** - Sobre compartilhamento
8. **Nao consentir** - E saber as consequencias
9. **Revogacao** - Do consentimento

---

## Principios da LGPD (Art. 6)

| Principio | Significado |
|-----------|-------------|
| **Finalidade** | Propositos legitimos e especificos |
| **Adequacao** | Compatibilidade com as finalidades |
| **Necessidade** | Minimo necessario para a finalidade |
| **Livre acesso** | Consulta facilitada sobre o tratamento |
| **Qualidade** | Dados exatos e atualizados |
| **Transparencia** | Informacoes claras e acessiveis |
| **Seguranca** | Medidas de protecao |
| **Prevencao** | Evitar danos |
| **Nao discriminacao** | Proibido tratamento discriminatorio |
| **Responsabilizacao** | Demonstrar conformidade |

## Obrigacoes do Controlador

### Documentacao
- Registro das atividades de tratamento (ROPA)
- Relatorio de Impacto (RIPD) quando necessario
- Politica de privacidade
- Politica de cookies

### Seguranca
- Medidas tecnicas e administrativas
- Controle de acesso
- Criptografia
- Backups
- Monitoramento

### Comunicacao
- Nomear Encarregado (DPO)
- Disponibilizar canal de atendimento
- Responder solicitacoes em ate 15 dias
- Comunicar incidentes a ANPD

---

## Transferencia Internacional

A transferencia de dados para outros paises so e permitida quando:

1. Para paises com nivel de protecao adequado
2. Mediante clausulas contratuais padrao
3. Mediante selos ou certificacoes
4. Com consentimento especifico do titular
5. Para cumprir obrigacao legal
6. Para execucao de contrato

---

## Sancoes (Art. 52)

| Sancao | Aplicacao |
|--------|-----------|
| **Advertencia** | Com prazo para correcao |
| **Multa simples** | Ate 2% do faturamento, limitado a R$ 50 milhoes por infracao |
| **Multa diaria** | Ate atingir o limite |
| **Publicizacao** | Divulgacao da infracao |
| **Bloqueio** | Dos dados pessoais |
| **Eliminacao** | Dos dados pessoais |
| **Suspensao parcial** | Do banco de dados por 6 meses |
| **Proibicao** | De atividades de tratamento |

## Estrutura de Politica de Privacidade

### Secoes Obrigatorias

1. **Identificacao do Controlador**
   - Razao social
   - CNPJ
   - Endereco
   - Contato do DPO

2. **Dados Coletados**
   - Categorias de dados
   - Forma de coleta
   - Dados obrigatorios vs opcionais

3. **Finalidades**
   - Para que cada dado e usado
   - Base legal para cada finalidade

4. **Compartilhamento**
   - Com quem e compartilhado
   - Motivo do compartilhamento

5. **Retencao**
   - Por quanto tempo dados sao mantidos
   - Criterios para eliminacao

6. **Seguranca**
   - Medidas adotadas
   - Criptografia, controle de acesso

7. **Direitos do Titular**
   - Lista dos direitos
   - Como exercer

8. **Cookies** (se aplicavel)
   - Tipos de cookies
   - Como gerenciar

9. **Contato**
   - Canal para solicitacoes
   - Prazo de resposta

10. **Atualizacoes**
    - Data da ultima atualizacao
    - Como serao comunicadas mudancas

## Checklist de Adequacao LGPD

### Governanca
- [ ] DPO nomeado e divulgado
- [ ] Politica de privacidade publicada
- [ ] Canal de atendimento ao titular
- [ ] Treinamento de equipe

### Documentacao
- [ ] Mapeamento de dados (ROPA)
- [ ] Base legal definida para cada tratamento
- [ ] Contratos com operadores atualizados
- [ ] Clausulas LGPD em contratos com terceiros

### Seguranca
- [ ] Controle de acesso implementado
- [ ] Criptografia em dados sensiveis
- [ ] Backup regular
- [ ] Plano de resposta a incidentes

### Consentimento
- [ ] Formularios com opcao clara de consentimento
- [ ] Registro de consentimentos
- [ ] Mecanismo de revogacao

### Direitos
- [ ] Processo para atender solicitacoes
- [ ] Prazo de resposta definido (max 15 dias)
- [ ] Formulario de solicitacao disponivel
