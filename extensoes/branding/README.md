# Extensao: Branding & Visual

Framework completo de branding, identidade visual e UI/UX para o Sleep Agent.

## Visao Geral

Esta extensao fornece agentes especializados, workflows completos e uma base de conhecimento extensa para criar marcas memoraveis e interfaces de alta conversao.

---

## Ativacao

```
/branding
```

---

## Agentes Disponiveis

### `/designer` - Designer Visual
Especialista em logo design, identidade visual e sistemas graficos.

**Especialidades:**
- Logo design (7 principios de Paul Rand)
- Paleta de cores
- Tipografia
- Elementos graficos
- Design systems

### `/uiux` - UI/UX Specialist
Especialista em interfaces, experiencia do usuario e landing pages.

**Especialidades:**
- Wireframes e prototipos
- Landing pages de alta conversao
- Design systems
- Especificacoes para dev
- Heuristicas de usabilidade

### `/brand` - Brand Strategist
Especialista em estrategia de marca, posicionamento e personal branding.

**Especialidades:**
- Golden Circle (Sinek)
- Brand Archetypes
- Positioning Statement
- Brand Voice
- Personal Branding

---

## Workflows

### `/brand-completo`
Processo completo de branding em 8 fases:
1. Descoberta
2. Estrategia
3. Identidade Verbal
4. Identidade Visual
5. Design System
6. UI/UX
7. Aplicacao
8. Personal Brand (opcional)

### `/landing-completa`
Criar landing page de alta conversao:
- Estrategia de conversao
- Copy persuasiva
- Wireframes
- Design
- Especificacoes

### `/identidade-visual`
Sistema de identidade visual:
- Conceituacao
- Logo
- Cores
- Tipografia
- Guidelines

## Exemplo de Uso

### Criar marca completa
```
> /brand-completo

Sleep Agent vai guiar voce pelas 8 fases:
1. Coletar briefing
2. Definir posicionamento
3. Criar identidade verbal
4. Desenvolver logo e visual
5. Sistematizar em design system
6. Criar interfaces
7. Aplicar em materiais
8. Posicionar fundador (opcional)
```

### Criar apenas logo
```
> /criar-logo

Designer vai:
1. Coletar informacoes da marca
2. Analisar concorrentes visuais
3. Aplicar principios de Paul Rand
4. Criar briefing completo para design
```

### Criar landing page
```
> /criar-landing

UI/UX vai:
1. Definir estrategia de conversao
2. Estruturar secoes (AIDA/PAS)
3. Escrever copy
4. Criar wireframes
5. Especificar para dev
```
