# Roteiros de Vídeo-Aulas - Base de Conhecimento

## Princípios Fundamentais

### 1. Atenção é Escassa
- Atenção média: 10-15 minutos
- Pico de atenção: primeiros 30 segundos
- Queda natural a cada 3-5 minutos

### 2. Vídeo ≠ Aula Gravada
Vídeo-aula não é gravar uma aula presencial. É um formato próprio.

### 3. Uma Ideia por Vídeo
Cada vídeo deve ter um objetivo claro e único.

---

## Estrutura de Vídeo-Aula

### Formato Padrão (5-10 minutos)

```
[0:00-0:30] HOOK
Capturar atenção com pergunta, estatística ou promessa

[0:30-1:00] ABERTURA
Tema, objetivo, por que importa

[1:00-7:00] CONTEÚDO
Blocos de conceito + exemplo

[7:00-8:30] RESUMO
Pontos principais (máximo 3)

[8:30-9:00] CTA
Próximo passo claro
```

### Formato Micro-Aula (3-5 minutos)

```
[0:00-0:15] HOOK
Frase de impacto

[0:15-0:30] OBJETIVO
O que vai aprender

[0:30-3:30] CONTEÚDO
Um conceito + um exemplo

[3:30-4:00] RESUMO
Uma frase síntese

[4:00-4:30] CTA
Ação imediata
```

### Formato Tutorial/Demo (10-15 minutos)

```
[0:00-0:30] HOOK
Resultado que vai alcançar

[0:30-1:30] CONTEXTO
Quando usar, pré-requisitos

[1:30-12:00] DEMONSTRAÇÃO
Passo a passo com explicação

[12:00-13:00] RECAPITULAÇÃO
Passos principais

[13:00-14:00] ERROS COMUNS
O que evitar

[14:00-15:00] CTA
Praticar por conta
```

## Elementos do Roteiro

### 1. Hook (Gancho)

**Função:** Capturar atenção nos primeiros segundos

**Tipos de Hook:**

```
PERGUNTA PROVOCATIVA
"Você sabia que 90% dos cursos online são abandonados?"

ESTATÍSTICA SURPREENDENTE
"Empresas que usam essa técnica aumentam vendas em 47%."

PROMESSA DIRETA
"Nos próximos 5 minutos, você vai aprender a..."

HISTÓRIA
"Maria tinha tudo para dar certo, mas cometeu um erro..."

DEMONSTRAÇÃO
"Olha o que acontece quando eu faço isso..."

PROBLEMA COMUM
"Se você já tentou fazer X e não conseguiu, esse vídeo é para você."
```

**Template:**
```
[HOOK - máximo 30 segundos]

(Escolha um tipo)

Pergunta: "Você já se perguntou por que [problema comum]?"
ou
Estatística: "[Número impactante] + contexto"
ou
Promessa: "Ao final deste vídeo, você será capaz de [resultado]"
```

---

### 2. Abertura

**Função:** Contextualizar e criar expectativa

**Elementos:**
- O que vai aprender (objetivo)
- Por que isso importa (relevância)
- O que precisa saber antes (pré-requisitos)

**Template:**
```
[ABERTURA - 30-60 segundos]

"Neste vídeo, você vai aprender [objetivo específico]."

"Isso é importante porque [razão relevante para o aluno]."

"Para acompanhar, você precisa [pré-requisito, se houver]."

"Vamos lá?"
```

---

### 3. Conteúdo

**Função:** Ensinar o conceito/habilidade

**Estrutura em Blocos:**

```
BLOCO 1: CONCEITO
- O que é
- Por que existe
- Quando usar

BLOCO 2: EXEMPLO
- Caso real ou ilustrativo
- Demonstração visual
- "Veja como funciona..."

BLOCO 3: ANTIEXEMPLO (opcional)
- Erro comum
- O que NÃO fazer
- "Muita gente erra ao..."

BLOCO 4: APLICAÇÃO
- Passo a passo
- "Agora é sua vez..."
- Template ou ferramenta
```

**Template de Bloco:**
```
[BLOCO X - 2-3 minutos]

Conceito: "[Nome/Ideia]"

"[Definição em uma frase]"

"Isso funciona porque [princípio/razão]."

"Por exemplo, [caso ilustrativo]."

[VISUAL: Mostrar exemplo]

"Na prática, você pode usar quando [contexto de aplicação]."

"O erro mais comum é [antiexemplo]. Evite isso porque [razão]."
```

---

### 4. Interações

**Função:** Manter atenção e verificar compreensão

**Tipos:**

```
PERGUNTA RETÓRICA
"Faz sentido até aqui?"

PAUSA PARA REFLEXÃO
"Pause o vídeo e pense: como você aplicaria isso?"

PREVISÃO
"O que você acha que vai acontecer se fizermos isso?"

QUIZ MENTAL
"Antes de eu mostrar, tente lembrar os 3 princípios."

CALL-TO-ACTION INTERMEDIÁRIO
"Anote isso, você vai precisar."
```

**Frequência:** A cada 2-3 minutos de conteúdo

---

### 5. Resumo

**Função:** Consolidar aprendizado

**Regra dos 3:** Máximo 3 pontos principais

**Template:**
```
[RESUMO - 30-60 segundos]

"Vamos recapitular o que você aprendeu:"

"Primeiro, [ponto 1 em uma frase]."

"Segundo, [ponto 2 em uma frase]."

"Terceiro, [ponto 3 em uma frase]."

"Lembre-se: [frase síntese de tudo]."
```

---

### 6. Call-to-Action (CTA)

**Função:** Direcionar próximo passo

**Tipos:**

```
EXERCÍCIO
"Agora faça o exercício X para praticar."

PRÓXIMA AULA
"No próximo vídeo, você vai aprender [teaser]."

APLICAÇÃO
"Escolha uma situação real e aplique o que aprendeu."

REFLEXÃO
"Responda nos comentários: como você vai usar isso?"

MATERIAL
"Baixe o template/checklist no link abaixo."
```

**Template:**
```
[CTA - 15-30 segundos]

"Seu próximo passo:"

"[Ação específica e realizável]"

"Se fizer isso, [benefício que vai obter]."

"Na próxima aula, [teaser do que vem]."

"Nos vemos lá!"
```

## Dicas de Escrita

### Linguagem

**Use:**
- Frases curtas (15-20 palavras)
- Voz ativa
- Palavras simples
- "Você" (conexão direta)

**Evite:**
- Jargão sem explicação
- Frases longas e complexas
- Voz passiva
- "Nós" ou impessoal

### Ritmo

```
RÁPIDO (energia)
- Hooks
- Transições
- CTAs

MÉDIO (clareza)
- Explicações
- Exemplos

LENTO (ênfase)
- Pontos importantes
- Momentos de reflexão
```

### Tom

```
Conversacional, não acadêmico.
"Imagine que você está explicando para um amigo."

Entusiasmado, não forçado.
"Mostre que você se importa com o tema."

Didático, não condescendente.
"Explique sem parecer que está falando com criança."
```

## Checklist de Roteiro

### Estrutura
- [ ] Hook nos primeiros 30 segundos
- [ ] Objetivo claro no início
- [ ] Blocos de 2-3 minutos
- [ ] Interações a cada 3-5 minutos
- [ ] Resumo com máximo 3 pontos
- [ ] CTA específico no final

### Conteúdo
- [ ] Uma ideia principal por vídeo
- [ ] Cada conceito tem exemplo
- [ ] Exemplos são relevantes para o público
- [ ] Erros comuns mencionados
- [ ] Aplicação prática clara

### Linguagem
- [ ] Frases curtas e diretas
- [ ] Linguagem conversacional
- [ ] Jargões explicados
- [ ] "Você" usado para conexão

### Visuais
- [ ] Descrição de cada visual necessário
- [ ] Momentos de B-roll identificados
- [ ] Gráficos/diagramas planejados

### Produção
- [ ] Duração estimada realista
- [ ] Materiais complementares listados
- [ ] Notas para edição incluídas
