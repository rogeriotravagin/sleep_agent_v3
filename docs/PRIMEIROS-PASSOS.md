# Primeiros Passos

Este guia vai te ensinar a usar o Sleep Agent em 5 minutos.

## 1. Ative uma Extensão

Primeiro, ative a extensão de Marketing:

```
/marketing
```

O Sleep Agent vai confirmar que a extensão foi carregada e mostrar os comandos disponíveis.

## 2. Faça seu Primeiro Pedido

Você pode fazer pedidos de duas formas:

### Forma Livre
Simplesmente descreva o que precisa:

```
Crie uma headline para meu curso de culinária
```

### Usando Comandos
Use comandos para tarefas específicas:

```
/criar-headline
```

O agente vai fazer perguntas para entender seu contexto e entregar o resultado.

## 3. Refine o Resultado

Não gostou? Peça ajustes:

```
Faça mais curta
```

```
Com mais urgência
```

```
Me dê mais 3 opções
```

## 4. Salve seu Trabalho

Quando gostar do resultado, salve:

```
/salvar
```

O arquivo será criado na pasta `projetos/`.

## Exemplos Práticos

### Criar Headlines

```
Você: Preciso de headlines para minha página de vendas

Sleep Agent: Entendido! Me conta:
1. O que você está vendendo?
2. Quem é seu público?
3. Qual o principal benefício?

Você: Vendo um curso de Excel para analistas de dados
      Público são profissionais que querem ser promovidos
      Benefício é dominar Excel avançado em 30 dias

Sleep Agent: [Entrega 5 headlines com explicação de cada uma]
```

### Criar Oferta

```
Você: /criar-oferta

Sleep Agent: Vamos construir uma oferta irresistível!
             [Faz as perguntas definidas na tarefa]

Você: [Responde]

Sleep Agent: [Entrega documento completo de oferta]
```

### Workflow de Lançamento

```
Você: /lancamento

Sleep Agent: Vamos preparar seu lançamento!
             Ao final você terá:
             - Oferta estruturada
             - Headlines
             - Sequência de emails
             - Conceitos de anúncios

             Me conta: o que você está lançando?

[Processo continua por todas as etapas]
```

## Dicas para Melhores Resultados

### Seja Específico

| Ruim | Bom |
|------|-----|
| "Crie um texto" | "Crie um email de vendas para meu curso de R$ 497" |
| "Headline para produto" | "Headline para curso de inglês, público 30-40 anos" |

### Dê Contexto

Quanto mais informação você der, melhor o resultado:

- Quem é seu público?
- Qual o preço?
- Qual o diferencial?
- Quais objeções comuns?

### Peça Variações

Não se contente com a primeira opção:

```
Me dê 5 variações
```

```
Agora com um ângulo diferente
```

### Use o Conhecimento

Os agentes usam bases de conhecimento. Você pode consultá-las:

```
Quais gatilhos mentais funcionam melhor para urgência?
```

### Para Começar
- `/marketing` - Ativa extensão de marketing
- `/ajuda` - Menu de ajuda

### Agentes
- `/copy` - Copywriter
- `/estrategia` - Estrategista

### Tarefas
- `/criar-oferta` - Oferta completa
- `/criar-headline` - Headlines
- `/criar-email` - Emails de venda

### Workflows
- `/lancamento` - Preparar lançamento
- `/campanha` - Criar campanha

### Utilitários
- `/salvar` - Salvar resultado
- `/status` - Ver status atual
