# Sleep Agent v4 Evolution - Progress Log

## Projeto
- **Nome:** Sleep Agent v4 - Evolucao Natural
- **Objetivo:** Evoluir o Sleep Agent com melhorias de memoria, contexto, qualidade e resiliencia
- **Essencia:** Manter foco em ajudar profissionais nao-tecnicos

## Codebase Patterns
<!-- Adicione padroes descobertos aqui -->
- Sleep Agent usa estrutura modular: core/ + extensoes/ + workspace/
- Extensoes sao plugins independentes com extensao.yaml como manifesto
- Agentes sao definidos em arquivos .md com persona, processo e referencias
- Tarefas usam formato YAML com entradas, passos, saida e gates
- Workflows sao sequencias de fases com agentes e tarefas
- Tom de voz: profissional mas acessivel, direto ao ponto, sem jargoes tecnicos

## Key Files Reference
- .claude/CLAUDE.md - Identidade principal do sistema

## Principios Criticos
1. **Nao perder essencia:** Sleep Agent e para profissionais NAO-TECNICOS
2. **Linguagem acessivel:** Evitar jargoes, ser direto
3. **Evolucao natural:** Melhorias parecem crescimento organico, nao importacao
