# Template de Progresso

Use este template para informar o usuário sobre o andamento de tarefas longas.

---

## Formato de Progresso

```
[Tarefa] Nome da Tarefa
[Status] Em andamento

Progresso:
[x] Passo 1 - Concluído
[x] Passo 2 - Concluído
[ ] Passo 3 - Em andamento
[ ] Passo 4 - Pendente

Próximo: Descrição do que vem a seguir
```

---

## Exemplo em Uso

```
[Tarefa] Criar Oferta Completa
[Status] Em andamento

Progresso:
[x] Entender o produto/serviço
[x] Definir público-alvo
[x] Identificar dores e desejos
[ ] Construir proposta de valor
[ ] Definir precificação
[ ] Criar garantia

Próximo: Vou agora construir a proposta de valor baseada nas dores identificadas.
```

---

## Quando Usar

- Tarefas com mais de 3 passos
- Workflows com múltiplas etapas
- Quando o usuário precisa acompanhar evolução

---

## Formato Simplificado

Para tarefas menores:

```
Passo 1 de 3: [Descrição] ✓
Passo 2 de 3: [Descrição] - em andamento
Passo 3 de 3: [Descrição]
```
